	Работа с разбиением:
- открыть приложение (все элементы в левом столбце неактивны, там стоят дефолтные значения; в поле справа дефолтное изображение, при щелчке можно загрузить новое; в верхнем меню активны кнопки база данных и вкл/выкл базу данных, добавить изображение, вкл/выкл сетку, неактивны кнопки загрузить изображение
- загрузить изображение нажатием на дефолтное изображение или на кнопку добавить изображение (становятся активными все элементы в левом столбце, поля Ширина и Высота заполняются дефолтными значениями в соответствии с пропорциями загруженного изображения, галка Пропорции стоит (если не стоит, то изображение сжимается в соответствии со значениями в полях Ширина и Высота и в них можно ввести независимые значения), поля Ширина фрагмента и Высота фрагмента заполняются дефолтными значениями, галка Квадрат стоит (если не стоит, в поля можно ввести независимые значения), при нажатии на кнопку Применить размеры применяются к изображению, при нажатии на кнопку Разбить разбиение применяется к изображению; в верхнем меню становится активной кнопка Загрузить изображение
- если Вкл база данных, то при разбиении изображения для окрашивания фрагментов используются цвета из базы данных; иначе используются средние цвета фрагментов
- если Загрузить изображение, то текущее изображение удаляется и загружается новое (всё по новой, см. предыдущий пункт)
- если Сохранить изображение, то текущее изображение сохраняется на ПК
- если Вкл сетка, то границы между фрагментами разбитого изображения явно выделены границами (сеткой); иначе сетки нет
	Работа с базой данных:
- база данных открывается нажатием кнопки База данных 
- в базу данных можно добавить новый цвет (выбор цвета из цветового круга, кнопка Добавить)
- из базы данных можно удалить цвет (выбор существующего цвета, кнопка Удалить)
